# GCP-ACAE
GCP Advanced Cloud Architect & AI Engineer

## Overview

This repository contains resources and preparation materials for the GCP Advanced Cloud Architect & AI Engineer certification exam. This advanced certification validates expertise in designing, implementing, and managing sophisticated cloud architectures with integrated AI/ML solutions on Google Cloud Platform.

## Target Profile

This certification is designed for experienced cloud professionals with the following background:

- **Prior Knowledge**: Already has Professional Cloud Architect (PCA) certification or equivalent knowledge
- **Experience**: 1â€“3 years of hands-on GCP infrastructure experience
- **Technical Skills**:
  - Proficient with **Terraform** for infrastructure as code
  - Comfortable with **Kubernetes (k8s)** for container orchestration
  - Experience with at least one **LLM stack** (Large Language Model frameworks and tools)
  - Understanding of cloud-native architectures and design patterns
  - Familiarity with AI/ML workflows and deployment practices

## Exam Structure

The exam consists of three main sections with the following weight distribution:

### 1. Hands-on Labs (60%)
The majority of the exam focuses on practical implementation skills:
- **Terraform**: Infrastructure provisioning and management
  - Writing and maintaining Terraform configurations
  - Managing state and modules
  - Implementing best practices for IaC
- **Kubernetes**: Container orchestration and management
  - Deploying and managing workloads on GKE
  - Configuring networking, storage, and security
  - Implementing autoscaling and monitoring
- **Vertex AI**: AI/ML model deployment and management
  - Training and deploying models
  - MLOps pipelines and automation
  - Integration with LLM services and frameworks

### 2. Architecture & Design (25%)
Architectural decision-making and system design:
- Designing scalable, resilient cloud architectures
- Cost optimization and resource management
- Security and compliance considerations
- Multi-region and hybrid cloud designs
- AI/ML pipeline architecture
- Integration patterns and best practices

### 3. Troubleshooting & SRE/Incident Response (15%)
Operational excellence and problem resolution:
- Debugging infrastructure and application issues
- Analyzing logs and metrics
- Implementing monitoring and alerting
- Incident response procedures
- Performance optimization
- Disaster recovery and business continuity

## Prerequisites

Before attempting this certification, ensure you have:

- Professional Cloud Architect (PCA) certification or equivalent knowledge
- Hands-on experience with GCP services and infrastructure
- Working knowledge of Terraform, Kubernetes, and at least one LLM framework
- Understanding of DevOps and SRE principles
- Familiarity with CI/CD pipelines and automation
